# nexoraAI

Modular Nexora/Xora assistant core now includes:

- `nexoraAI/orchestrator.py`: intent routing + multi-agent orchestration
- `nexoraAI/decision_engine.py`: context/permission/risk/autonomy decisioning (levels 0-3)
- `nexoraAI/memory_engine.py`: short/mid/long memory with structured knowledge extraction
- Agent modules:
  - `memory_agent.py`
  - `research_agent.py`
  - `calendar_agent.py`
  - `wellness_agent.py`
  - `communication_agent.py`
  - `notification_agent.py`
  - `security_agent.py`
  - `device_agent.py`
